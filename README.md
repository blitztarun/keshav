## About

This project is a React application written in TypeScript, demonstrating typed component development, structured state management, and clean front-end architecture.

---

## Repository structure

```
keshav/
├── app.tsx          # Root application component
├── index.tsx        # Entry point
├── index.html       # HTML shell
└── metadata.json    # App configuration/metadata
```

---

## Running locally

```bash
# Clone the repo
git clone https://github.com/blitztarun/keshav.git
cd keshav

# Install dependencies
npm install

# Start the development server
npm start
```
